## Overview

PrestaCMSMediaBundle integrates Sonata Media in [PrestaCMS][1].

This bundles adds the following block type to your project :

-   Media Simple
-   Media Advanced
-   Gallery Simple
-   Gallery Advanced
-   Carousel
